fill_holes_nest

For a input binary mask in NESTED ordering, fill_holes_nest flip to 1 all pixels located in invalid regions with fewer pixels than the threshold provided.
Two pixels are adjacent (and belong to the same region or hole) if they have at least one point in common.

Location in HEALPix directory tree: src/f90/mod/mask_tools.F90 

FORMAT

call fill_holes_nest( nside, new_min_size, mask_in, mask_out )

ARGUMENTS

name & dimensionality kind in/out description
       
nside I4B IN The Nside value of the input mask.
new_min_size I4B IN Minimal size of hole (in pixels) on output
mask_in(0:Npix-1) I4B IN Input NESTED-ordered mask. Npix = 12*nside*nside
mask_out(0:Npix-1) I4B OUT Output NESTED-ordered mask. Can be the same array as mask_in.

EXAMPLE:

use healpix_types
use healpix_modules
...
call fill_holes_nest(nside, new_min_size, mask_in, mask_in)
???
