nside2npix

Function to provide the number of pixels over the full sky corresponding to resolution parameter .

Location in HEALPix directory tree: src/C/subs/chealpix.c 

FORMAT

long nside2npix(const long nside)

ARGUMENTS

name&dimensionality kind in/out description
       
nside long IN the parameter of the map.
nside2npix long OUT returns the number of pixels of the map .

EXAMPLE:

npix= nside2npix(256);
Returns the number of HEALPix pixels (786432) for the resolution parameter 256.
