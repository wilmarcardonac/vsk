plsfci: Set FCI (font characterization integer)

plsfci ( fci );
 

Sets font characteristics to be used at the start of the next string using the FCI approach. See the section called “FCI” for more information.

fci (PLUNICODE, input)

PLUNICODE (unsigned 32-bit integer) value of FCI.
