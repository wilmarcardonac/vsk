The PDF Driver

A basic version of a pdf driver has been added to PLplot. This driver is based on the libharu library see: libharu.org. At present only the Hershey fonts are used and there is no support for pdf or ttf fonts. Compression of the pdf output is not enabled and the paper size can't be chosen. All these issues will be addressed in later releases.
