plgyax: Get y axis parameters

plgyax (p_digmax,
 p_digits);
 

Identical to plgxax, except that arguments are flags for y axis. See the description of plgxax for more detail.

p_digmax (PLINT *, output)

Pointer to variable with the maximum number of digits for the y axis. If nonzero, the printed label has been switched to a floating point representation when the number of digits exceeds p_digmax.

p_digits (PLINT *, output)

Pointer to variable with the actual number of digits for the numeric labels (y axis) from the last plot.

Redacted form: plgyax(p_digmax, p_digits)

This function is used in example 31.
